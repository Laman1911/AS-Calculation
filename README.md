Alpha Solutions Project Calculator

Dette projekt er udviklet som en del af KEA 2. semester eksamensprojektet.

Formål
Applikationen skal hjælpe med at beregne og administrere projektdata for Alpha Solutions' konsulentopgaver.

Teknologier
Java 17
Spring Boot
Thymeleaf
JPA (Hibernate)
MySQL (eller H2 til udvikling)
Arkitektur
Projektet er bygget efter MVC-principper:

Model: Data- og entitetslag
View: Thymeleaf frontend
Controller: Forretningslogik og routing
